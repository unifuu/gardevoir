## 2K25
### Dec 27
> 오히려 새로운 그림에 어울리는 풍경이나, 사람들이 쉽게 놓치는 아름다운 경치를 발견하곤 하니까요.
> 
> → むしろ、新しい絵にふさわしい風景や、人々が見落としがちな美しい景色を見つけることがよくありますから。

| KR    | JA         |
| ----- | ---------- |
| 오히려   | むしろ        |
| 어울리는  | ふさわしい      |
| 풍경이나  | 風景や        |
| 쉽게    | 簡単に        |
| 놓치다   | 見逃す / 見落とす |
| -곤 하다 | -たりする      |
| -니까요  | -からです      |
