### Dec 28
> 그리고 그런 발견들이 항상 제 영감이 되기도 하고요... 죄송해요... 그러니까, 사실 제가 정말 하고 싶은 말은, 지금 햇빛에 비친 방랑자의 모습과 그림자가 너무 아름답다는 거예요...! 이 순간을... 그림에 담아도 될까요?

### Dec 27
> 오히려 새로운 그림에 어울리는 풍경이나, 사람들이 쉽게 놓치는 아름다운 경치를 발견하곤 하니까요.
> 
> → むしろ、新しい絵にふさわしい風景や、人々が見落としがちな美しい景色を見つけることがよくありますから。

| KR    | JA         | CH  |
| ----- | ---------- | --- |
| 오히려   | むしろ        |     |
| 어울리는  | ふさわしい      |     |
| 풍경    |            | 風景  |
| -이나   | -や         |     |
| 쉽게    | 簡単に        |     |
| 놓치다   | 見逃す / 見落とす |     |
| 아름다운  | 美しい        |     |
| 경치    |            | 景致  |
| -곤 하다 | -たりする      |     |
| -니까요  | -からです      |     |
